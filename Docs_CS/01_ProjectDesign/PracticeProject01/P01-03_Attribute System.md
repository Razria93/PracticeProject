# 구성요소
### 속성 종류
1. 불
2. 물
3. 땅
4. 바람

### 시스템 개요
- 각 속성은 상성을 가지고 있음
	- 상성일 경우 1.5배 데미지
	- 역상성일 경우 0.5배 데미지
	- 무상성일 경우 1.0배 데미지
	- 상성표는 다음과 같음

| 속성 이름 | 상성 (1.5배) | 역상성 (0.5배) |
| :---: | :-------: | :--------: |
|   불   |    바람     |   물 / 땅    |
|   물   |     불     |     바람     |
|   땅   |     물     |     바람     |
|  바람   |   물 / 땅   |     불      |
