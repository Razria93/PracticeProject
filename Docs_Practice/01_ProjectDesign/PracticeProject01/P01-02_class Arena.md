# 구성요소
### 데이터
1. 몬스터의 최대 등록 가능 수
2. 몬스터 컨테이너 (배열)

### 기능
1. 몬스터를 추가하는 함수 (최대 등록 가능 수 이하)
2. 현재 들어있는 몬스터의 수를 가져오는 함수
3. 생명력이 가장 높은 몬스터를 조회할 수 있는 함수
4. 현재 몬스터의 상태를 조회 할 수 있는 함수
	1. 이름
	2. 생명력
	3. 공격력
	4. 방어력
	5. 속성

5. GoToAround 함수
	- 해당 함수 호출 시 다음과 같은 알고리즘이 실행됨
		1. 아레나의 추가된 순서대로 정렬 후 각 객체 별 타겟을 설정
		2. 몬스터들이 순차적으로 타격
		3. 생명력이 0이 될 경우 사망하고 해당 아레나에서 제외 됨
6. StartGame 함수
	- 몬스터가 한마리 남을 때까지 GoToAround를 동작
	- 마지막 남은 몬스터를 우승자로 출력
